## 登録しているスニペット一覧

### `getid`: Get Element by id

```javascript
const name = document.getElementById('');
```

### `getcl`: Get Elements by className

```javascript
const name = document.getElementsByClassName('');
```

### `qs`: Query Selector

```javascript
const name = document.querySelector('.class');
```


```javascript
const name = () => {};
```
